# Installing RD Mobile

## Requirements
- Android 10+ with Bluetooth
- Unknown apps installation enabled

## Installation
1. Receive APK via Bluetooth from desktop
2. Open the APK file on Android
3. Allow installation from unknown sources
4. Install the app

## Permissions
- Bluetooth: For pairing and communication
- Location: For Bluetooth scanning (optional)

## First Run
- App displays "RD Mobile Running"
- Wait for bootstrap from desktop
- UI updates to "Bootstrap Complete" after pairing
