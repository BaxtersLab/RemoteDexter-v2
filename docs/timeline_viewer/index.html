Filtering

      
Type in the Filter box to highlight matching rows on the current page.

      
Global Filter: when enabled, the filter scans the entire dataset and highlights rows on the current page that match anywhere in the full trace.

      
Pagination

      
Use Page Size, Prev/Next and Go controls to navigate. Sorting preserves page offsets (no reshuffle of page boundaries).
