# Installing RD Desktop

## Requirements
- Windows 10/11 with Bluetooth
- Paired Android device

## Installation
1. Download rd-desktop.exe
2. Run the executable (no installation needed)
3. Follow on-screen menu

## Usage
1. Pair your Android device via Windows Bluetooth settings
2. Run rd-desktop.exe
3. Select 1 to discover devices
4. Select 2 to bootstrap Android (transfers APK)
5. Install APK on Android
6. Select 3 for handshake
7. Select 4 to send ping

## Bluetooth Pairing
- Enable Bluetooth on both devices
- Go to Windows Settings > Devices > Bluetooth
- Add device and confirm pairing
