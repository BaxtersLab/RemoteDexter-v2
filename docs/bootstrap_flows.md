# Bootstrap Flows

Bootstrap is an offline, user-approved process using a Bootstrap Channel.

## Bluetooth bootstrap
- User enables Bluetooth on both devices.
- User approves pairing on both devices.
- RD Mobile Application is installed or verified.

## AirDrop bootstrap
- User initiates transfer on the User PC.
- User approves the transfer on the User Device.

## USB bootstrap
- User connects the User Device to the User PC.
- User approves any OS-level prompts.

## Safety boundaries
- No silent installs
- No hidden pairing steps
- User confirmation required for every bootstrap step
- Bootstrap does not grant remote access without an explicit pairing and session start
